QuIP - QUick Image Processing

An interactive environment for imaging and scientific computing

The QuIP interpreter is a powerful interactive environment for image processing and scientific computing. It features integration with a variety of video interfaces, GPU computing with both nVidia's CUDA and OpenCL. The system is written mostly in C, with help from yacc/bison and the m4 macro processor. It runs on unix-like systems such as linux and Mac OSX, and has been ported to iOS.

How to get QuiP

The source code for QuIP can be obtained as a tar archive, ready to build with the traditional configure/make/install sequence, or by cloning the git repository located at https://github.com/jbmulligan/quip.

Documentation

The QuIP manual is written in the texinfo system, and is available here as either single PDF file, or on-line web pages.

Contact

QuIP was written and is maintained by Jeff Mulligan. Early versions were developed while a student at UCSD in the mid 80's, although very little of that original code can be found in the current sources, which were developed at NASA Ames Research Center.
